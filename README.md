# vue-ip-tracker

[Live Demo](https://vue-ip-tracker-live.netlify.app/)


## Project set up
- ```git clone https://github.com/SebastienLcmt/vue-ip-tracker.git```
- ```npm install```
- ```npm run serve```


## Project installation memo

- ```vue create projectName```
- ```cd projectName```
- ```npm i leaflet```
- ```npm i axios```
